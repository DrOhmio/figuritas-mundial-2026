# Mi Álbum Mundial 2026 v2.0

Aplicación PWA para controlar figuritas, faltantes, repetidas, respaldos y comparación con amigos.

## Publicar en GitHub Pages
1. Borra o reemplaza los archivos actuales del repositorio.
2. Sube todo el contenido de esta carpeta conservando las subcarpetas.
3. Confirma el commit.
4. Espera uno o dos minutos y abre GitHub Pages.

## Datos existentes
La app conserva la misma clave de almacenamiento local usada por la versión anterior:
`wc26_pro_v6_portable`

Por eso, el progreso existente debería mantenerse en el mismo navegador y dispositivo.

## Compartir
Cualquier persona puede abrir el mismo enlace y tendrá su propio progreso local. Para transferir datos debe usar Exportar respaldo e Importar respaldo.
